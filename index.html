Face Data Processing (Important)

What Face Data We Collect

We do not collect or store any face data on our servers.

Face data in this app refers only to images (photos, GIFs, or video frames) that users manually select within the app.

We do NOT collect or generate biometric identifiers such as facial recognition templates, embeddings, or any persistent identity data.

How Face Data Is Used

Face data is used solely for providing face swap functionality, including:

  • Detecting faces in images or GIFs
  • Aligning facial features
  • Generating face-swapped results

All processing is performed entirely on the user’s device.

Local Processing (No Upload)

All face processing is done locally using on-device AI models.

No images, face data, or biometric data are uploaded to any server.

Face Data Sharing

We do not share face data with any third parties.

No third-party APIs are used for face processing.

Face Data Retention

Face data is not retained by the app.

  • Processing occurs in memory only
  • Data is deleted immediately after processing
  • No face data is stored unless the user explicitly saves the result to their device

No Facial Recognition

This app does not perform identity recognition, authentication, or tracking of individuals.
